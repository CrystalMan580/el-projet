Basic Linux Commands: Debian vs. Arch

you can see the apps you need in linux here

btw if you using ubunto , mint , kali , or any other debian based distro u can use the same commands



Command Description Debian Arch Action
Update Package List Updates the list of available packages and their versions. sudo apt update sudo pacman -Sy
Upgrade Installed Packages Upgrades all installed packages to the latest version. sudo apt upgrade sudo pacman -Su
Install a Package Installs a specified package. sudo apt install package_name sudo pacman -S package_name
Remove a Package Removes a specified package. sudo apt remove package_name sudo pacman -R package_name
Search for a Package Searches for a package in the repository. apt search package_name pacman -Ss package_name
Show Package Information Displays information about a specific package. apt show package_name pacman -Qi package_name
Open a folder Open a specific directory. cd 'folder name' cd 'folder name'
Back to Previous Folder Go back to the previous folder. cd .. cd ..
Monitor System Performance Real-time monitoring of system resources. htop htop
View System Logs Displays system logs for troubleshooting. journalctl -xe journalctl -xe
Check Disk Usage Shows disk usage of file systems. df -h df -h
Find Files Searches for files in the directory hierarchy. find /path/to/search -name "filename" find /path/to/search -name "filename"
Change File Permissions Changes the permissions of a file or directory. chmod 755 filename chmod 755 filename
Change File Ownership Changes the owner and/or group of a file. chown user:group filename chown user:group filename
Network Configuration Displays network interfaces and configurations. ip addr show ip addr show
Package Cleanup Removes unused packages and dependencies. sudo apt autoremove sudo pacman -Rns $(pacman -Qdtq)
Backup and Restore Creates a backup of files/directories. tar -czvf backup.tar.gz /path/to/backup tar -czvf backup.tar.gz /path/to/backup
