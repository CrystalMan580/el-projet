Heres freeBSD basic commands

Command Description
ls Lists directory contents.
cd Changes the current directory.
cp Copies files and directories.
mv Moves or renames files and directories.
rm Removes files or directories.
mkdir Creates a new directory.
man Displays the manual for a command.
pkg Package management tool for installing and managing software.
pkg install <package> Installs a package.
pkg update Updates the package repository catalog.
pkg upgrade Upgrades all installed packages to their latest versions.
shutdown Shuts down or reboots the system.
top Displays active processes and system resource usage.
free Displays free and used memory in the system.
df Displays disk space usage for file systems.
du Estimates file and directory space usage.
ifconfig Displays or configures network interfaces.
ping Checks network connectivity to another host.
firewall Configures the FreeBSD firewall (ipfw, pf).
